# Agrinho_2026_valeria
Esse jogo é um mini simulador de agricultura sustentável feito em JavaScript com p5.js, onde o jogador controla um agricultor que precisa irrigar flores para fazê-las crescer. O objetivo principal é regar todas as flores até que elas atinjam o crescimento máximo, mostrando uma mensagem de vitória no final.

O jogador controla um agricultor que pode se mover para a esquerda e para a direita usando as setas do teclado. Ele também pode apertar a tecla espaço para lançar gotas de água. Essas gotas caem em linha reta e servem para irrigar as flores.

No campo existem dez flores alinhadas. Cada flor começa pequena e vai crescendo conforme recebe água. Sempre que uma gota atinge uma flor, ela aumenta seu tamanho e o jogador ganha pontos. As flores são coloridas aleatoriamente, deixando o jogo mais visualmente variado.

O cenário do jogo é simples e animado, com céu azul, sol, nuvens que se movem lentamente, um gramado verde e um moinho de vento girando continuamente, criando um ambiente de fazenda vivo e agradável.

Quando todas as flores atingem o crescimento máximo, o jogador vence o jogo e aparece uma tela de vitória com a mensagem “FAZENDA SUSTENTÁVEL!”. Nessa tela também é possível reiniciar o jogo pressionando a tecla R, o que faz os pontos voltarem a zero, recria as flores e reinicia a partida.

Em resumo, é um jogo simples de movimentação e precisão, onde o jogador interage com o ambiente através da irrigação, mostrando de forma divertida a ideia de cuidado com a natureza e crescimento sustentável.
